Introduction

Python is a high-level, interpreted, general-purpose programming language that emphasizes readability and simplicity. It was created by Guido van Rossum and released in 1991. Python allows developers to express their ideas in fewer lines of code compared to other programming languages. It is dynamically typed and garbage-collected, meaning that memory management is handled automatically, and you do not need to declare the data types of variables explicitly.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming. It also has a large standard library and supports a vast ecosystem of third-party libraries that make it suitable for many tasks, from web development and automation to data science and machine learning.

What you should already know

This documentation assumes you have the following background:

  • A general understanding of how computers and programming work: Familiarity with how a computer processes code, executes commands, and interacts with hardware and software.
  • Basic knowledge of working with text files and operating systems: Understanding how to open, write, and read files, as well as basic system operations.
  • Familiarity with basic programming concepts: Concepts such as loops, conditionals, and variables will be helpful. If you're new to programming, consider starting with an introductory Python tutorial or course.

If you're new to programming, there are many introductory tutorials available for Python to help get you started.

Variables

A variable in Python is a symbolic name that is bound to a value. Variables do not need an explicit declaration of type. The type of the variable is determined when the program runs.

x = 42 name = "John"

Variables in Python are assigned using the = operator. Python does not use type declarations like int, float, or str; instead, it infers the type based on the value assigned.

Declaring variables

In Python, variables are created by assigning values to them. Python does not require an explicit declaration statement or type information. This is because Python is dynamically typed.

x = 10 name = "Python"

Once a variable is assigned a value, Python will automatically determine its type.

x = 10 y = 3.14 message = "Hello"
